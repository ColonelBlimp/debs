Double Entry Bookkeeping System (DEBS)
====
**Status**<br/>

**Building**<br/>

Modify the maven _settings.xml:_

    <profile>
      <id>dev</id>
	   <properties>
	     <dev.webappDirectory>[path-to-webappDir]</dev.webappDirectory>
	  </properties>
	</profile>

Don't forget to make the profile active.
